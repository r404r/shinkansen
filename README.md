# Shinkansen 🚄

快速、流暢的 Chrome 網頁翻譯擴充功能。使用 Google Gemini 將外語網頁翻譯成繁體中文（台灣用語），採單語覆蓋模式——原地替換文字，保留頁面原始排版。

名稱「新幹線」象徵快速、平穩、流暢的閱讀體驗。

## 功能特色

- **單語覆蓋翻譯**：原地替換網頁文字，保留元素的字型、字級、顏色與排版
- **漸進式翻譯**：分批送出、逐批注入，頁面逐段變成中文，不必等全部翻完
- **佔位符系統**：自動保留連結、粗體、斜體等行內元素結構，翻譯後原樣還原
- **全文術語表一致化**：長文翻譯前先擷取專有名詞對照表，確保全文譯名統一
- **翻譯快取**：相同原文只翻一次，跨頁面共用快取，節省 API 費用
- **三維 Rate Limiter**：RPM / TPM / RPD 滑動視窗，自動適配 Gemini API 配額
- **用量追蹤**：IndexedDB 記錄每次翻譯的 token 數與費用，附圖表與 CSV 匯出
- **編輯譯文**：翻譯完成後可直接在頁面上修改譯文
- **還原原文**：一鍵切換回原文，隨時對照

## 安裝方式（開發版）

1. 開啟 Chrome，網址列輸入 `chrome://extensions/` 並按 Enter
2. 右上角打開「開發人員模式」
3. 點「載入未封裝項目」（Load unpacked）
4. 選擇本專案的 `shinkansen/` 資料夾
5. 擴充功能清單會出現 Shinkansen，可以固定到工具列

## 首次設定

1. 點工具列的 Shinkansen 圖示 → 「設定」
2. 貼上你的 Gemini API Key（取得：<https://aistudio.google.com/apikey>）
3. 預設模型 `gemini-3-flash-preview`、Service Tier `DEFAULT`
4. 其餘參數可依需求調整（溫度、每批段數、字元預算等）

## 使用方式

- **手動翻譯**：點工具列圖示 → 「翻譯本頁」
- **快捷鍵**：`Option + S`（macOS）/ `Alt + S` 切換翻譯 / 還原
- **自動翻譯**：在設定頁白名單加入網域，進入該網站自動翻譯
- **術語表**：在 Popup 或設定頁開啟「術語表一致化」，長文翻譯會先建立專有名詞對照表
- **編輯譯文**：翻譯完成後，在 Popup 點「編輯譯文」可直接修改頁面上的譯文

## 自動化測試

本專案使用 Playwright 建立了回歸測試系統，涵蓋段落偵測、佔位符序列化、注入路徑等核心邏輯。測試使用 synthetic HTML fixture 與 canned LLM 回應，不需呼叫 Gemini API，約 20 秒跑完全部測試。

```bash
npm test                    # 全部測試（22 條）
npm run test:regression     # 只跑回歸測試（14 條）
npm run test:unit           # 只跑單元測試（7 條）
npm run test:version        # 只跑版本號檢查（1 條）
```

詳細操作說明見 `測試流程說明.md`。
