# Shinkansen-Nozomi 🚄

[繁體中文](README.md) | [日本語](README.ja.md)

高速でスムーズなウェブページ翻訳ブラウザ拡張機能。Chrome と Firefox に対応。Google Gemini を使って外国語ページを繁体字中国語／簡体字中国語／日本語に翻訳し、テキストをその場で置き換え、ページの元のレイアウトを保持します。

名前は新幹線最速の「のぞみ号」に由来し、高速・安定・スムーズな読書体験を象徴しています。

> **本プロジェクトは主に Firefox 拡張機能に注力しています。Chrome 拡張機能については原作者の [jimmysu0309/shinkansen](https://github.com/jimmysu0309/shinkansen) をご参照ください。**

> **本プロジェクトは [jimmysu0309/shinkansen](https://github.com/jimmysu0309/shinkansen) からフォーク**（原作者：Jimmy Su）し、Firefox 対応・多言語インターフェース（繁中／簡中／日本語）・選択範囲翻訳などの機能を追加して独立開発しています。

> [最新版 zip をダウンロード](https://github.com/r404r/shinkansen/releases/latest)

## このプロジェクトの目的

既存の没入型翻訳ツールは、個人の閲覧内容をサードパーティのサーバーに送信する必要があり、プライバシーの管理が難しい状況です。Shinkansen-Nozomi は設計の最初からプライバシーを核心に据えています。すべての設定とデータはお使いのコンピュータにのみ保存され、ご自身で用意した Gemini API Key で Google に直接接続する以外、いかなるデータも第三者に送信しません。ソースコードは完全公開されており、誰でもその安全性を検証できます。

## 主な機能

- **ウェブページのレイアウトを保持**：テキストをその場で置き換え、フォント・サイズ・色・レイアウトを保持。リンクはクリック可能、太字・斜体もそのまま
- **単一言語 / バイリンガル対照の2モード**（v1.5.0〜）：ポップアップでワンクリック切替。4種類の視覚マーク（薄い背景色 / 左側バー / 波線下線 / なし）
- **段階的翻訳**：バッチ送信・逐次注入。ページが段落ごとに翻訳され、全体の完了を待つ必要なし
- **3つの翻訳エンジン**：Gemini（AI翻訳・最高品質）+ Google Translate（無料・高速）+ **カスタムモデル**（v1.5.7〜、OpenAI互換エンドポイント、OpenRouter / DeepSeek / Ollama等に対応）
- **3組のカスタマイズ可能なショートカット**：`Alt+A` / `Alt+S` / `Alt+D` にそれぞれ翻訳プリセット（エンジン+モデル+ラベル）を割り当て
- **選択範囲翻訳**（v1.6〜）：テキストを選択してショートカットを押すと、選択範囲内の段落のみ翻訳。ページ全体は翻訳しない
- **YouTube字幕リアルタイム翻訳**：YouTube字幕を自動検出し、リアルタイムで翻訳。字幕スタイルはYouTubeネイティブと完全一致
- **固定用語集**：翻訳対照表をカスタマイズ。グローバルとドメイン専用の2層をサポート
- **禁止用語リスト**（v1.5.6〜）：翻訳結果に含まれてはならない用語の対照表
- **翻訳キャッシュと即時費用レポート**：2層キャッシュメカニズム（ローカルキャッシュ + Gemini implicit cache）
- **3次元レートリミッター**：RPM / TPM / RPD スライディングウィンドウ
- **使用量追跡**：各翻訳のトークン数と費用を記録。グラフとCSVエクスポート付き
- **訳文編集**：翻訳完了後、ページ上で直接訳文を修正可能
- **タブ間翻訳継続**（v1.4.11〜）：タブAで翻訳後、リンクから新しいタブBを開くとBも自動翻訳
- **指定サイト自動翻訳**：設定ページでドメインを追加すると、そのサイトを開いた際に自動翻訳
- **原文復元**：同じショートカットキーで原文に戻す

## インストール方法

**Firefox — 開発版（一時的な読み込み）**

1. Firefox を開き、アドレスバーに `about:debugging#/runtime/this-firefox` と入力
2. 「一時的なアドオンを読み込む」（Load Temporary Add-on）をクリック
3. `build/firefox/manifest.json` を選択（事前に `npm run build:firefox` を実行）
4. 注意：一時的に読み込んだ拡張機能は Firefox の再起動後に消えます

**Chrome — 開発版（パッケージなしで読み込み）**

1. Chrome を開き、アドレスバーに `chrome://extensions/` と入力
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `shinkansen/` フォルダ（または `build/chrome/` ビルド成果物）を選択

## 初期設定

1. Gemini API Key を申請 — 詳細は [API Key 申請ガイド](API-KEY-SETUP.md) を参照
2. ツールバーの Shinkansen-Nozomi アイコン → 「設定」をクリック
3. Gemini API Key を貼り付け
4. デフォルトモデル `gemini-3-flash-preview`、Service Tier `DEFAULT`
5. その他のパラメータは必要に応じて調整

## インターフェース言語設定

Shinkansen-Nozomi は3つのインターフェース言語に対応。翻訳の対象言語とプロンプトも同時に切り替わります：

| 言語 | 翻訳対象 | プロンプトの文脈 |
|------|---------|-----------------|
| 繁体字中国語（デフォルト） | 台湾繁体字中国語 | 台湾の用語・通用訳名 |
| 簡体字中国語 | 大陸簡体字中国語 | 大陸の標準用語・通用訳名 |
| 日本語 | 日本語 | 自然な日本語表現・日本の通用訳名 |

切替方法：設定 → 一般 → インターフェース言語

## 使い方

- **手動翻訳**：ツールバーアイコン → 「このページを翻訳」
- **翻訳ショートカット**（v1.4.12〜 3組）：
  - `Option+A`（macOS）/ `Alt+A` — デフォルト Gemini Flash Lite（低コスト）
  - `Option+S` / `Alt+S` — デフォルト Gemini Flash（高品質、おすすめ）
  - `Option+D` / `Alt+D` — デフォルト Google Translate（無料、APIキー不要）
  - 翻訳済み状態でショートカット → 原文に戻す
  - 翻訳中にショートカット → 翻訳キャンセル
- **選択範囲翻訳**（v1.6〜）：テキストを選択してショートカットを押すと、選択範囲の段落のみ翻訳
- **YouTube字幕翻訳**：英語字幕付き動画を開き、CCをオンにして、ポップアップから「YouTube字幕翻訳」をオン
- **指定サイト自動翻訳**：設定ページの「自動翻訳ウェブサイト」にドメインを追加
- **固定用語集**：設定ページの「用語集」タブで対照エントリを追加
- **訳文編集**：翻訳完了後、ポップアップで「訳文を編集」をクリック
